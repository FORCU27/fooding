# Design System Guide

- **버전**: DS v1.2 (업데이트 시 변경)
- **범위**: 컬러/타이포/그리드/컴포넌트 규칙, 접근성 가이드.

## Tokens
- 컬러: `Primary 500 #F74A27`, `Neutral 900 #1D1D1D`, `Success 500 #12B76A`.
- 타입: Pretendard / 4px 베이스 라인 / 모듈 스케일 1.2.
- 간격: 4px 단위, Card/Section 패딩 16px/24px 기준.

## Components
- Core 컴포넌트 카탈로그는 `frontend/guides/component-library.md`와 연동.
- 상태(hover/active/disabled)와 모션 사양을 Figma Variants로 정의하고 문서화.

## 접근성
- 대비비율 AA 이상, 텍스트 14px 이상.
- 모션 민감도 대응: Reduce Motion 설정 시 페이드/슬라이드 최소화.
