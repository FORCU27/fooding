# Fooding Partner

| Field | Detail |
| --- | --- |
| **Primary Target** | 외식업 사장님, 식자재/설비 공급사 |
| **Mission** | 도매가 식자재·설비·서비스를 Fooding 생태계 안에서 연결 |
| **Business Model** | 거래액 기반 수수료(5~10%), 구독형 혜택, 정기배송/후불 결제 |

## Strategic Role
- Fooding POS/CEO 사용자 기반에 안정적인 공급망을 제공해 **양면 네트워크 효과**를 강화합니다.
- 데이터 기반 추천(재고/메뉴/계절)으로 매장 운영 의사결정을 돕습니다.
- 파트너 입점, 카탈로그 관리, 정산을 자동화해 입점 장벽을 낮춥니다.

## Partner Journey
1. **입점 신청**: 파트너 센터에서 카테고리, 납품 지역, MOQ 등록.
2. **상품 등록**: SKU, 단가, 패키지, 리드타임, 프로모션 설정.
3. **거래/정산**: 매장 주문 접수 → 배송 추적 → 정산/수수료 처리.
4. **관계 관리**: 리뷰/평가, CS, 프로모션 성과 리포트.

## Core Capabilities
- **마켓플레이스**: 검색/필터, 추천 묶음(식재, 장비, 서비스), 대량/정기 구매 옵션.
- **정산/재무**: 후불 결제, 정기 결제, 세금계산서 발행, 수수료 자동화.
- **입점 지원**: 온보딩 체크리스트, 카탈로그 CSV 업로드, SLA 모니터링.
- **데이터 인사이트**: POS 판매 데이터 기반 추천, 재고 소진 예측, 공동 구매 제안.

## KPIs
- 월간 거래액(GMV), 활성 파트너 수, 매장당 평균 주문 금액, 정기배송 유지율.
- 주문→배송 완료 리드타임, CS 응답시간, 파트너 만족도.

## Dependencies
- Commerce/Order 서비스, Payment/정산 시스템, Logistics/배송 추적, Notification.
- Data Warehouse(판매/재고/리워드), Partner CRM, Elasticsearch(카탈로그 검색).

## Status & Next Steps
- ✅ Partner 마켓 기획, B2B 혜택 정의, 서비스 모델 초안 완성.
- 🔄 파트너 온보딩 플로우, 정기배송/후불 결제 MVP, 공급사 대시보드.
- 📌 향후에는 Fooding Place, CEO 대시보드와 파트너 추천을 상호 연동해 교차 판매를 넓힐 예정입니다.
