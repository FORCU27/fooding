# Fooding Place

| Field | Detail |
| --- | --- |
| **Primary Target** | 매장/브랜드 마케터 |
| **Core Promise** | 노코드 웹사이트 빌더로 Fooding 예약/리뷰와 즉시 연동되는 스토어 프레즌스 제공 |
| **Tech Stack** | Zapp 기반 노코드 엔진 + Tailwind 템플릿 |

## Positioning
- SNS/배달 플랫폼에 의존하지 않고 **자체 사이트**를 소유할 수 있도록 지원합니다.
- Fooding 예약, 웨이팅, 리뷰 데이터를 위젯 형태로 퍼블리싱해 **실시간 신뢰도**를 보여줍니다.
- 검색/SEO 최적화, 메뉴/이벤트 업데이트를 코딩 없이 빠르게 반영할 수 있습니다.

## Feature Pillars
- **템플릿/테마**: 업종별 프리셋, 색상/폰트 커스터마이즈, 반응형 미리보기.
- **예약/웨이팅 위젯**: Fooding API 연동, 실시간 좌석/대기 숫자 표시, CTA 추적.
- **콘텐츠 블록**: 메뉴, 스토리, 리뷰 하이라이트, 미디어 갤러리, 지도/주차 정보.
- **SEO/배포**: 메타 태그, OG 이미지, 자동 사이트맵, 커스텀 도메인 연결.
- **확장성**: 파트너 마켓 상품 노출, 이벤트/공동 프로모션 삽입, 알럿 배너.

## KPIs
- 구축 사이트 수, 템플릿 전환율, Fooding 예약 위젯 클릭률, SEO 유입, CTA → 방문 전환.
- CMS 업데이트 소요 시간, site publish 실패율.

## Dependencies
- Builder 엔진(Zapp), Media/S3 저장소, Reservation/Waiting API, Review API, SEO 파이프라인.
- CMS 권한/버전 관리, CDN 캐시, Analytics 태깅 시스템.

## Roadmap
- ✅ Place 확장 기획 완료, 예약/사이트 통합 고도화 계획 수립.
- 🔄 템플릿 마켓, 이미지 최적화 워크플로, 번역/다국어 지원.
- 📌 장기적으로 Fooding Partner 상품 위젯, AI 카피 제안, 메뉴 추천 블록을 제공할 예정입니다.
